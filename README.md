# Todo

A simple todo application built with React and Vite.

## Features

- Add new todos
- Mark todos as complete
- Delete todos
- Edit todos
- Show all todos
- Responsive design

## Technologies Used

- JavaScript
- React
- Vite
- Tailwind CSS

## Live Demo

You can view the live demo of the application [here](https://12sushree.github.io/todo/).

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/12Sushree/todo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd todo
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:5173` to see the website.

## Usage

- Add a new todo by typing in the input field and pressing Enter.
- Click on a todo to mark it as complete.
- Click on the delete button to remove a todo.
- Use the edit button to change the content of a todo.
